@() CRUD REST API using Play

CRUD REST API using Play

To see a list of customers (3 per page), you can do a GET: 

        http GET localhost:9000/v1/customers/list?page=1

To see a particular customer's info, you can do a GET: 

        http GET localhost:9000/v1/customers/:id

To create new customers, do a POST: 

        http POST localhost:9000/v1/customers name="name" location="location"

To update a customer's info, do a PUT: 

        http PUT localhost:9000/v1/customers name="new name" location="location"

To delete a customer's info, do a DELETE: 

        http DELETE localhost:9000/v1/customers/:id
